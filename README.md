# ReplaceValorantBGVideo
![png](https://github.com/Anderson413/ReplaceValorantBGVideo/blob/main/program.png)

## 怎么用？
1.更换国服背景视频只需要点击自动检测然后选择当前赛季的背景视频
<br>点击“选择视频”旁边的“...”选择你想替换的视频
<br>然后启动瓦罗兰特就可以自动替换了

## 注意！
1. 打开软件之前请确保游戏完全关闭 否则无法替换

2. 目前“自动检测”功能仅支持国服，点击它会自动定位背景视频所在的位置	点击旁边的“...”按键然后选择本赛季的背景视频文件（不选就更改不了背景视频）
<br>**注：国际服也能换，但是要你自己手动点击“...”按键选择国际服的背景视频文件**

3. 推荐使用16:9的视频，要是用其他比例视频的话 游戏会自动拉伸！

4. 选择好要被替换的视频文件之后后，点击“加入”加入到视频列表
   然后双击你想要替换上去的视频，最后打开游戏

5. 请勿在游戏运行时关闭软件 这将不利于游戏关闭时恢复视频文件的功能！
